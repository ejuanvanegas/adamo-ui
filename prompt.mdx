Create the combobox stories.

Make sure to refer to: 
- shadcn/ui docs: https://ui.shadcn.com/docs/components/combobox
- radix ui docs: https://www.radix-ui.com/primitives/docs/components/combobox

IMPORTANT rules:
- Add meta.title like Components/ComponentName, for example Components/Button
- Do not add meta.parameters.layout centered
- Use the correct components and tailwind prefixed classes
- Do not add (args types)/controls, just the stories
- All imports must be absolute like @src/components/...
- Use proper story name case, only first word capitalized (name: With icon), the component/(story itself) can be named normally (WithIcon), for example:

export const WithIcon: Story = {
  name: "With icon",
  render: function Render() {
    return (
      <Field>
        ...
      </Field>
    );
  },
};

- Do not import React directly, instead import what is needed, for example, useState instead of importing all React
- Add descriptions to all stories
- Render the component correctly in the stories using the render function
- Name the stories files in kebab-case
- Setup the component props using the args parameter in the story itself, and destructure the args in the render function, for example (if args needed):

export const Default: Story = {
  args: { type: "single", collapsible: true },
  render: function Render(args) {
    return (
      <Accordion
        {...args}
        className={cn("adm:w-full")}
      >
      </Accordion>
    );
  },
};
